# Deep Dive Template

```
[AI Security Deep Dive] YYYY-MM-DD — TITLE

Source: LINK (author, date)

## Summary
2–3 sentence overview of the paper, report, or technique.

## Attack / Technique
- What it does
- How it works (simplified)
- Affected systems or models

## Defense / Mitigation
- Proposed countermeasures
- Current limitations

## Implications
Why this matters for practitioners, vendors, or policy.

## Key Takeaway
One sentence distilling the most important point.
```

## Instructions

- Focus on a single paper, report, or technique.
- Keep the attack description accurate but accessible.
- Always include defense/mitigation — never present attacks without context.
